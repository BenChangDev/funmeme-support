# FunMeme 技术支持 / 常见问题

## 常见问题

1. **如何内购积分？**
   - 在App的【支付/购买】页面选择所需积分包即可，支持支付宝/微信/苹果内购。

2. **购买未到账怎么办？**
   - 请确认网络畅通，未到账请截图并通过邮箱联系我们。

3. **如何联系客服？**
   - 邮箱、App内留言或微信均可。
